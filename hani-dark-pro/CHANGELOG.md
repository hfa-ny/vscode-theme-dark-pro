# Change Log

All notable changes to the **HFA Dark Pro** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-05-01

### Added
- Initial release of HFA Dark Pro
- Dark UI theme based on Atom One Dark
- Tuned syntax highlighting for JS/TS, React (JSX/TSX), PHP, Blade, HTML, CSS
- Custom activity bar, sidebar, and editor background palette
