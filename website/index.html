Install in 30 seconds

  1. Open VS Code
  2. Press Ctrl+Shift+X (or ⌘+Shift+X)
  3. Search Eloquent Dark Pro and click Install
  4. Press Ctrl+Shift+P → Preferences: Color Theme → pick Eloquent Dark Pro
install.sh 
# or install from the command line
code --install-extension eloquentsoftware.eloquent-dark-pro
