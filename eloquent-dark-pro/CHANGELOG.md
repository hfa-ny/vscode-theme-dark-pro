# Change Log

All notable changes to the **Eloquent Dark Pro** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-01

First polished release under the **Eloquent Software** publisher.

### Added
- New publisher identity: `eloquentsoftware`
- Proper marketplace listing: icon, gallery banner, screenshots, README
- Promo site at the project repository
- MIT license
- Expanded keyword set (laravel, typescript) for better discoverability

### Changed
- Renamed theme from "HFA Dark Pro" to **Eloquent Dark Pro**
- Refined README with installation steps and recommended editor settings
- Tightened `.vscodeignore` so packages no longer ship `.git`, source maps, or stray `.vsix` files

### Notes
- The original unpolished build was used by hundreds of developers — this release
  is the same theme they already love, with the documentation and presentation it deserved from day one.
